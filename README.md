# Nazbil Design

[MDN](https://developer.mozilla.org/en-US/docs/Web/HTML)에 있는 HTML elements을 UI/UX에 적합하게 콤포넌트화 했습니다.
Basic과 Application으로 나누어 졌습니다. 실제 사용은 Application을 사용하시면 됩니다. React 18.3.1, Typescript 4.9.5로 제작 했습니다.
